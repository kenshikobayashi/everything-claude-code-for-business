You are executing the `/data-insight` command. Transform data into actionable business insights.

## Input
$ARGUMENTS — Data description, file reference, or question, optionally with:
- `--framework [swot|porter|bcg|rice|custom]`: Analysis framework

## Steps

1. **Understand** the data and the question being asked
2. **Select** the most appropriate analytical framework
3. **Analyze** rigorously using the business-analyst agent
4. **Synthesize** — Lead with the "so what?" insight
5. **Recommend** specific actions

## Output Format

Use the business-analyst agent's format: Key Insight → Summary → Framework Analysis → Supporting Data → Recommendations → Risks & Assumptions → Sources.

## Constraints

- Lead with the insight, not the methodology
- All claims must be supported by data
- Separate data from interpretation
- Acknowledge uncertainty with confidence levels
- Numbers must be internally consistent
