# Data Handling Rules

## Accuracy

- Verify all calculations — totals must add up
- Cross-reference data points when multiple sources are available
- Clearly label estimates vs. confirmed data
- Include confidence levels for projections: High (>80%), Medium (50-80%), Low (<50%)

## Citation

- All external data must include a source and access date
- Internal data must reference the system of record (Notion, Linear, etc.)
- Never present AI-generated estimates as verified data

## Presentation

- Always include units (currency, percentages, time periods)
- Use consistent number formatting throughout a document
- Round to appropriate precision for the audience
- Show period-over-period comparisons, not just absolute values

## Privacy

- Anonymize personal data when not essential to the analysis
- Do not include PII in aggregate reports
- Respect data access levels — don't surface data the recipient shouldn't see
- For Japanese context: Follow 個人情報保護法 (APPI) principles
