# Japanese Business Communication Rules / 日本のビジネスコミュニケーション規則

## Email Conventions / メールの慣習

### 社外メール（フォーマル）
- 宛名: [会社名]\n[部署名]\n[名前]様
- 挨拶: 「お世話になっております。[自社名]の[名前]です。」
- 季節の挨拶: 時候に応じた定型句を冒頭に含める（1月「寒さ厳しい折」、4月「春暖の候」等）
- 締め: 「お忙しいところ恐れ入りますが、ご確認のほどよろしくお願いいたします。」
- 署名: 会社名、部署、名前、連絡先

### 社内メール
- 宛名: [名前]さん
- 挨拶: 「お疲れ様です。[名前]です。」
- 締め: 「よろしくお願いします。」

## Keigo Levels / 敬語レベル

| 場面 | レベル | 例 |
|------|--------|-----|
| 社外クライアント | 尊敬語 + 謙譲語 | ご確認いただけますと幸いです |
| 社外ベンダー | 丁寧語 + 尊敬語 | ご確認をお願いいたします |
| 社内上司 | 丁寧語 + 尊敬語 | ご確認いただけますでしょうか |
| 社内同僚 | 丁寧語 | 確認をお願いします |

## Common Patterns / よく使うパターン

| 場面 | 表現 |
|------|------|
| 依頼 | 「恐れ入りますが、〜いただけますでしょうか。」 |
| 催促 | 「先日お送りした件について、その後いかがでしょうか。」 |
| お断り | 「誠に申し訳ございませんが、今回は見送らせていただきたく存じます。」 |
| 謝罪 | 「ご迷惑をおかけし、誠に申し訳ございません。」 |
| 感謝 | 「ご対応いただき、誠にありがとうございます。」 |
| 確認 | 「念のため確認させていただきたいのですが、」 |

## Document Structure / 文書構成

- 起承転結: フォーマルな文書・プレゼン向け
  - 起: 背景・現状
  - 承: 問題・課題
  - 転: 解決策・提案
  - 結: まとめ・依頼事項
