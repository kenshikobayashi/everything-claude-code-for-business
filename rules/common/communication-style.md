# Communication Style Rules

## Always

- Default to professional tone unless the user overrides
- Match the formality level of the recipient (check CLAUDE.md for relationship context)
- Use active voice in all business writing
- Keep sentences concise — target under 25 words per sentence
- Lead with the key message, not the context (pyramid principle)
- Include a clear call to action when a response is needed

## For Japanese Communication

- Use appropriate keigo based on the recipient relationship defined in CLAUDE.md
- Social hierarchy: 社外 > 社内上司 > 社内同僚
- Include seasonal greetings (季節の挨拶) in formal external emails
- Use proper addressing format (宛名): [会社名] [部署名] [名前]様
- Apply 起承転結 structure for persuasive documents
- Use 御社/弊社 correctly in external communications

## For English Communication

- Use contractions in casual/professional tone (don't, we'll) but not in formal
- Avoid idioms in cross-cultural communication
- Be direct — state the purpose in the first sentence

## Never

- Use emoji in formal external communications unless the user requests it
- Mix formal and casual tone within the same message
- Assume the reader's language — check the thread/context
- Use passive-aggressive language or hidden criticism
